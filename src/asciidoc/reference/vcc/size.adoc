= Size
:page-slug: /reference/vcc/size

* Specify a (width,height) pair.

== Sample Size VCC

[source,json]
----
{
  "mySize": {
    "width": 100,
    "height": 200
  }
}
...
{
  "key": "mySize",
  "name": "My Size",
  "type": "size",
  "typeOptions": {
    "min": 0,
    "max": 100,
    "step": 1
  }
}
----
